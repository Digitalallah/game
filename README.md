# Пока гроза

«Пока гроза» — браузерная 2D arcade platformer game на HTML Canvas без npm, сборщиков, сервера и базы данных. Игра запускается как обычный сайт, совместима с GitHub Pages и адаптирована для Telegram Mini App.

## Запуск

Откройте `index.html` в браузере или опубликуйте репозиторий через GitHub Pages. Все ассеты подключаются относительными путями, поэтому отдельная сборка не нужна.

## Персонажи

Перед стартом доступен выбор персонажа:

- `Кот` — существующий составной персонаж из `assets/player/`;
- `Пес` — составной персонаж из `assets/Player2/` при наличии файлов `body_sob.png`, `head_sob.png`, `left-hand_sob.png`, `right-hand_sob.png`, `left-leg_sob.png`, `right-leg_sob.png` или `righ-leg_sob.png`, `tail_sob.png`.

`full_sob.png`, если добавлен в папку ассетов, используется только как референс сборки и не накладывается поверх игрового персонажа. Выбор сохраняется в `localStorage`, а физика, хитбокс, скорость, прыжок, гравитация, жизни и баланс у кота и пса одинаковые — отличается только внешний вид и голова в HUD.

## Управление

Клавиатура на компьютере:

- `A` / `D` или `ArrowLeft` / `ArrowRight` — движение;
- `W`, `Space`, `ArrowUp` — прыжок;
- `P` или `Escape` — пауза;
- `R` после поражения — начать заново.

На сенсорных устройствах отображаются кнопки `◀`, `▶`, `⤴`, а в нижней строке — `II ПАУЗА` и кнопка звука. Сенсорная панель учитывает safe area iPhone и скрывается на обычном ПК через проверку `navigator.maxTouchPoints > 0` и `matchMedia("(pointer: coarse)")`.

## Игровые механики

- Собирайте обычных рыбок для очков и золотых рыбок для повышенного бонуса.
- Избегайте предупреждающих зон молний: активная молния строится процедурно сверху вниз, без отдельного ассета.
- Если уйти из опасной зоны почти перед ударом, начисляется бонус `+25` без текстового сообщения.
- Крабы наносят ровно одну жизнь урона, после чего включается временная неуязвимость.
- Молния может уничтожить краба. Тогда всегда выпадает сердечко: оно лечит при неполных жизнях или даёт `+50` очков при полном запасе.

## Рост сложности и антикемпинг

Сложность зависит только от активного времени текущего раунда: пауза, скрытая вкладка и свернутый Mini App не увеличивают таймер сложности. Очки на сложность не влияют.

В игре используется единый плавно растущий коэффициент: со временем молнии появляются чаще, иногда идут сериями, предупреждение немного сокращается, крабы появляются быстрее, становятся быстрее, чаще прыгают и могут выходить на платформы. Рост не имеет видимых стадий и не останавливается полностью, но отдельные параметры ограничены, чтобы опасности оставались читаемыми.

Скрытый антикемпинг отслеживает стояние в одной области и на одной платформе. Если игрок почти не меняет позицию около 2,5–3 секунд, постепенно растет шанс молнии рядом с зоной ожидания, краба на маршруте и ценной рыбки в другом месте. При активном перемещении давление быстро снижается.

## Типы крабов

- Обычный краб идет по земле от края к краю.
- Прыгающий краб заранее приседает и поднимает клешни, затем прыгает по читаемой дуге.
- Платформенный краб может запрыгивать на платформы и продолжать маршрут без телепортации.
- Суетливый краб на высокой сложности иногда останавливается, разворачивается и идет дальше.

Все крабы удаляются после выхода за границы или по ограниченному времени жизни, поэтому массивы объектов не растут бесконечно.

## Telegram Mini App

В `index.html` подключён официальный Telegram Web App SDK. При запуске игра безопасно проверяет `window.Telegram?.WebApp`, вызывает `ready()` и `expand()`, если API доступен. В обычном браузере игра продолжает работать без Telegram API.

При сворачивании Mini App или скрытии вкладки игра ставится на паузу, а активное время раунда не увеличивается.

## Публикация результата

На экране поражения есть кнопка `ПОДЕЛИТЬСЯ РЕЗУЛЬТАТОМ`. Она формирует Telegram share URL с текстом:

`Ого, я набрал XXX очков в игре «Пока гроза»! Попробуешь побить мой рекорд?`

Ссылка Mini App задаётся константой `MINI_APP_URL` в `src/main.js`. Замените значение `https://example.github.io/game/` на фактический GitHub Pages или Telegram Mini App URL проекта.

Если доступен Telegram Mini App API, публикация открывается через `Telegram.WebApp.openTelegramLink()`. В обычном браузере используется `window.open()`.

## Звук

Звуковые эффекты и тихая эмбиент-мелодия создаются через Web Audio API без внешних аудиофайлов и библиотек. Один переиспользуемый `AudioContext` создаётся или возобновляется только после пользовательского действия.

Есть эффекты рыбок, золотой рыбки, прыжка, приземления, молнии, урона, сердечка, кнопок интерфейса и попадания молнии в краба. Кнопка `ЗВУК ВКЛ` / `ЗВУК ВЫКЛ` отключает одновременно музыку и эффекты, применяет плавную громкость и сохраняет состояние в `localStorage`.

## Структура проекта

```text
index.html              # точка запуска игры и Telegram SDK
styles.css              # адаптивная вёрстка, стартовый экран и сенсорные кнопки
src/main.js             # логика игры, рендер, Telegram API, звук, пауза и управление
assets/back.png         # фон
assets/player/          # части персонажа «Кот»
assets/Player2/         # ожидаемые части персонажа «Пес»
assets/crab_*.png       # части краба
assets/fish*.png        # обычные и золотые рыбки
```

Canvas и CSS используют pixelated/crisp rendering, а в игровом контексте сохраняется `ctx.imageSmoothingEnabled = false`.
