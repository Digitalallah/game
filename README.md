# Котогроза

«Котогроза» — простая браузерная 2D arcade platformer game на HTML Canvas без внешних библиотек, сборщиков и npm.

## Как запустить

Откройте файл `index.html` в любом современном браузере. Игра работает локально и не требует сервера.

## Управление

- `ArrowLeft` / `A` — идти влево
- `ArrowRight` / `D` — идти вправо
- `Space` / `ArrowUp` / `W` — прыжок
- `P` — пауза / продолжить
- `R` — перезапустить игру

## Правила

У кота 3 жизни. Собирайте рыбок, выживайте под дождём и уворачивайтесь от молний. Перед ударом молнии 1 секунду мигает зона предупреждения. Если забрать рискованную рыбку прямо перед ударом, появится надпись «РИСК!» и начислится большой бонус.
