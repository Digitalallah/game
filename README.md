# Котогроза

«Котогроза» — браузерная 2D arcade platformer game на HTML Canvas без npm, сборщиков, сервера и базы данных. Игра запускается как обычный сайт и адаптирована для Telegram Mini App.

## Как запустить локально

Откройте файл `index.html` в современном браузере. Все пути к стилям, скриптам и изображениям относительные, поэтому проект работает и локально, и из подпапки GitHub Pages, например `/game/`.

## Управление

### Клавиатура

- `ArrowLeft` / `A` — идти влево
- `ArrowRight` / `D` — идти вправо
- `Space` / `ArrowUp` / `W` — прыжок
- `P` — пауза / продолжить
- `R` — перезапустить игру

### Сенсорные кнопки

Для телефона и Telegram Mini App добавлены полупрозрачные экранные кнопки в нижней части экрана:

- `◀` — идти влево;
- `▶` — идти вправо;
- `⤴` — прыжок.

Кнопки используют Pointer Events, поэтому работают с мышью, стилусом и сенсорным экраном. Одновременное нажатие направления и прыжка поддерживается.

## Telegram Mini App

В `index.html` подключён официальный Telegram Web App SDK:

```html
<script src="https://telegram.org/js/telegram-web-app.js?62"></script>
```

При запуске игра безопасно проверяет `window.Telegram?.WebApp`, вызывает `ready()` и `expand()`, если API доступен. В обычном браузере игра продолжает работать без Telegram API.

Также добавлены безопасные haptic feedback-вызовы для Telegram:

- обычная рыбка — лёгкий отклик;
- рискованный бонус — успешное уведомление;
- урон — ошибочное уведомление;
- близкий удар молнии — сильный отклик.

При сворачивании Mini App или скрытии вкладки игра ставится на паузу. После возвращения остаётся экран «ПАУЗА», и таймер с движением молний не продолжаются, пока приложение скрыто.

## GitHub Pages

1. Загрузите репозиторий на GitHub.
2. Убедитесь, что в корне есть пустой файл `.nojekyll`.
3. Откройте **Settings → Pages**.
4. В блоке **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку с игрой, обычно `main`, и папку `/root`.
6. Сохраните настройки и дождитесь публикации.
7. Откройте выданный URL вида `https://<user>.github.io/<repo>/`.

Проект не использует абсолютные пути вида `/assets/...`, поэтому ресурсы корректно загружаются из подпапки репозитория.

## Подключение URL к Telegram-боту

1. Откройте диалог с [@BotFather](https://t.me/BotFather).
2. Выберите вашего бота командой `/mybots`.
3. Откройте **Bot Settings → Menu Button**.
4. Выберите **Configure menu button**.
5. Введите текст кнопки, например `Играть`.
6. Введите URL опубликованной GitHub Pages-версии, например `https://<user>.github.io/game/`.
7. Сохраните настройки и откройте бота в Telegram на телефоне.

Для отдельной Mini App-кнопки также можно использовать BotFather-разделы, связанные с Web Apps / Mini Apps, и указать тот же HTTPS URL.

## Как тестировать Mini App

- Откройте GitHub Pages URL в обычном браузере и проверьте, что стартовый экран, кнопка «ИГРАТЬ», клавиатура и сенсорные кнопки работают без Telegram API.
- Откройте тот же URL через кнопку бота в Telegram и проверьте запуск Mini App.
- Нажмите «ИГРАТЬ» только после загрузки — на мобильном раунд не начинается сам.
- Проверьте вертикальный экран телефона: canvas должен сохранять квадратные пропорции, не обрезать HUD, кота и нижнюю сцену.
- Проверьте, что нет горизонтальной и вертикальной прокрутки, а жесты не масштабируют страницу во время игры.
- Сверните Telegram или переключитесь на другое приложение, затем вернитесь: должен остаться экран «ПАУЗА».
- Проверьте сбор рыбок, урон и близкий удар молнии в Telegram: при доступности API должен быть аккуратный haptic feedback.

## Правила

У кота 3 жизни. Собирайте рыбок, выживайте под дождём и уворачивайтесь от молний. Перед ударом молнии мигает зона предупреждения. Если забрать рискованную рыбку прямо перед ударом, появится надпись «РИСК!» и начислится большой бонус.

## Структура проекта

```text
index.html              # точка запуска игры и Telegram SDK
styles.css              # мобильная адаптация, safe area и сенсорные кнопки
src/main.js             # логика игры, рендер, Telegram API, пауза и управление
assets/                 # изображения фона и персонажа
.nojekyll               # отключает Jekyll на GitHub Pages
README.md               # описание проекта
```
