# Котогроза

«Котогроза» — браузерная 2D arcade platformer game на HTML Canvas без внешних библиотек, сборщиков и npm.

Игра сохранена как пиксельный 8-bit arcade: ночная гроза, мокрая городская улица, дождь, отражения, фонари, силуэты домов и маленький белый кот в оранжевом дождевике с зонтиком.

## Как запустить

Откройте файл `index.html` в любом современном браузере. Игра работает локально и не требует сервера.

## Управление

- `ArrowLeft` / `A` — идти влево
- `ArrowRight` / `D` — идти вправо
- `Space` / `ArrowUp` / `W` — прыжок
- `P` — пауза / продолжить
- `R` — перезапустить игру

## Правила

У кота 3 жизни. Собирайте рыбок, выживайте под дождём и уворачивайтесь от молний. Перед ударом молнии мигает зона предупреждения. Если забрать рискованную рыбку прямо перед ударом, появится надпись «РИСК!» и начислится большой бонус.

## Визуальные улучшения

- Обновлён общий стиль под цельную 8-bit / pixel-art arcade сцену с ночной грозой и мокрым асфальтом.
- Главный персонаж теперь читается как белый кот с большими оранжевыми очками, оранжевым дождевиком, сапогами и зонтиком.
- Добавлены простые покадровые состояния персонажа:
  - `idle` — спокойное ожидание;
  - `run` — бег с перестановкой сапог;
  - `jump` — прыжок;
  - `fall` — падение;
  - `hurt` — реакция на удар;
  - `collect` — короткая радостная реакция на бонус.
- Улучшены молнии: предупреждающий луч и зона удара, мягкая вспышка экрана, яркий разряд, свечение и отражение на мокрой поверхности.
- Фон стал слоистым: тучи, дождь, дальний город, силуэты зданий, фонари, мокрая улица и платформы как городские навесы / балки / крыши.
- Бонусы получили пульсацию, свечение для рискованной рыбки, вспышки-частицы и pop-up очков при подборе.
- Русский HUD сохранён и оформлен в пиксельные панели: «ЖИЗНИ», «СЧЁТ», «РЕКОРД», «ВРЕМЯ», «ПАУЗА».

## Структура проекта

```text
index.html      # точка запуска игры
styles.css      # стили страницы и pixel-perfect отображение canvas
src/main.js     # логика игры, рендер, эффекты и анимации
README.md       # описание проекта
```

Папка `assets/` не требуется: вся графика на этом этапе отрисовывается кодом в Canvas, чтобы сохранить запуск напрямую через `index.html` без сборщиков и сторонних библиотек.
